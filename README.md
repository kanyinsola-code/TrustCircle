# TrustCircle
AI-powered group savings platform for GTCO Squad Hackathon 3.0
# The Problem
In traditional Ajo, when a member defaults, the next person suffers. Coordinators use Excel and there is no safety net, no credit tracking, and no automation. Trust is easily broken.
# Our Solution
TrustCircle automates the process using Squad's APIs. We hold the funds securely in a Squad virtual account, automate the payout when it's your turn, and use an AI Trust Score to build financial reputation. Our Emergency Swap feature ensures the circle never breaks if someone can't pay.

## Key Features
- **Automated Ajo:** Auto-collects contributions and pays the next person on schedule via Squad.
- **AI Trust Score:** Tracks member reliability, building a financial reputation for everyone.
- **Emergency Swap:** Allows a trusted member to swap payout positions to cover a default.
- **Penalty Box:** Small automated late fees that compensate the waiting member.
- **Voice-Guided:** Built for accessibility so visually impaired users can participate.
- **Hybrid Tracking:** Supports both bank transfers and offline cash payments.

## Thematic Areas (Hackathon)
This project covers all four focus areas for the Hackathon:
1.  **AI Automation:** Automating collections, payouts, and default handling.
2.  **Predictive AI Tools:** The Trust Score predicts future reliability.
3.  **Squad API Integration:** Core functionality relies on Squad's Virtual Accounts & Transfers.
4.  **Financial Innovation:** Transforming the traditional Nigerian Ajo into a smart system.

## Tech Stack
- **Frontend:** React / Next.js
- **Backend:** Node.js / Express
- **API:** Squad (Virtual Accounts, Transfers)
- **Database:** Firebase / MongoDB
